Features

5 Themed Levels

Jungle Dawn, Lava Cavern, Ice Peaks, Storm Ruins, Shadow Abyss — each with unique weather effects, obstacles, and increasing difficulty.

Online Multiplayer

Host a room, share a 6-character code. Friends join from anywhere. Host approves players before the game starts. Up to 4 players.

Animal Enemies

Cats chase you on the ground. Dogs jump over obstacles. Dragons shoot fireballs every 2 seconds. Shoot them or escape.

5 Playable Characters

Boy, Girl, Grandpa, Grandma, Officer — each visually unique. Enter your name and pick your hero on the character select screen.

Survival Mechanics

3 lives, HP bar, energy bar that drains over time. Collect hearts to restore health and ammo. Shield pickups for invincibility.

Gun Combat

Press F or left-click to shoot. 12 bullets per life shown in the HUD. Gunshot sound, glowing bullet trail, and kill scores.

Weather Effects

Rain in Jungle, lightning in Storm, snowflakes in Ice, rising embers in Lava, purple mist in Abyss.

World Map

Parchment-style level select map. Levels unlock as you progress. Hover to see difficulty info before entering.

How to Run

1
Download Python 3.11+ from python.org if you don't have it.
2
Download the game ZIP using the button above, then extract it.
3
Open a terminal in the extracted folder and run:
pip install -r requirements.txt
4
Start the game:
python -B game.py
